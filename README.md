# 🎮 Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Challenge the computer and test your luck! ✨

---

## 🚀 Live Demo

🔗 **Play Now:**
https://kallupallianusha.github.io/Rock-Paper-Scissors-Game/

---

## ✨ Features

✔️ Play against the computer
✔️ Randomized computer choices
✔️ Instant result display
✔️ Clean and responsive UI
✔️ Beginner-friendly project

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling
* **JavaScript** – Game logic

---

## 📂 Run Locally

Follow these steps to run the project on your system:

1. Clone the repository

   ```bash
   git clone https://github.com/KallupalliAnusha/Rock-Paper-Scissors-Game.git
   ```
2. Open the project folder
3. Double-click **index.html**

---

## 🎯 How to Play

1. Choose one option:

   * ✊ Rock
   * ✋ Paper
   * ✌️ Scissors
2. The computer will make a random choice
3. The result (Win/Lose/Draw) will be displayed instantly
